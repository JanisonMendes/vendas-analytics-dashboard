# 📊 Sales Performance Dashboard

(Dashboard.png)

## 🎯 Objetivo
Transformar dados brutos de vendas em insights estratégicos para a tomada de decisão sobre categorias de produtos e performance regional.

## 🛠️ Processo Técnico
1. **Extração:** Dados extraídos via PostgreSQL (scripts na pasta `/sql`).
2. **Transformação:** Tratamento de dados e criação de medidas no Excel.
3. **Visualização:** Design de dashboard em Dark Mode com foco em UX.

## 📈 Principais Indicadores (KPIs)
* Receita Total e Ticket Médio.
* Performance por Categoria e Região.
* Perfil Demográfico dos Clientes.
